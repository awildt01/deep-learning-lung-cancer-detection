# deep-learning-lung-cancer-detection
Lung Cancer Detection with Deep Learning (LUNA16)  Dieses Projekt implementiert eine Deep-Learning-Pipeline zur Klassifizierung von Lungenknoten in CT-Scans auf Basis des LUNA16-Datensatzes. 3D-Ausschnitte um Kandidatenpunkte werden mit einem 3D-CNN als Knoten oder Nicht-Knoten klassifiziert.
